# Health-Care
Description: This project is an interactive healthcare analytics dashboard built using Power BI to provide insights into hospital operations, patient trends, and financial performance. The dashboard integrates 8 key tables â€” Cities, Departments, Diagnoses, Insurance, Patients, Procedures, Providers, and Visits.
Key Features:

Financial KPIs: Total Billing Amount, Total Treatment Cost, Total Medication Cost, Total Room Charges, Insurance Coverage, Out-of-Pocket Expenses.

Patient Insights: Total patients, average billing per visit, patient distribution by city, age group, and gender.

Operational Insights: Average Length of Stay, department-wise and procedure-wise utilization, emergency visits, and follow-up analysis.

Interactive Visuals: Drill-downs by city, department, diagnosis, and procedure; slicers for date, service type, and insurance provider.

Advanced DAX Measures: Calculated KPIs including averages, percentages, and ratios for better decision-making.

Objective:
To provide hospital management with actionable insights for financial planning, resource allocation, and patient care optimization.

Tools Used:
Power BI, DAX, Data Modeling, Star Schema, SQL, Excel.

Use Case:

Identify high-revenue departments and cost-intensive procedures.

Track patient flow and resource utilization across locations.

Evaluate insurance coverage and out-of-pocket expenses.

Support data-driven operational and financial decisions.
